    advanced data structure

    preamble:A graduate course in Computer Applications should give due exposure to the recent developments. Since Data structures is a central pillar of any program on Computer Science/ Applications, this course is designed to build upon the knowledge acquired at the undergraduate level and familiarise students with a bunch of modern data structures which are quite useful to solve, in the most effective manner, the modern, real life problems.

    Course Outcomes:
    1. Remember the Basic Data Structures and understand the Set Data Structure and its implementation.
    2. Understand Advanced Heap Structures suitable for solving Computational problems involving Optimisation and analysing these data structures using amortised analysis.
    3. Understand the basic operation of Blockchaining along with the data structures used in it and the challenges in Blockchain data.
